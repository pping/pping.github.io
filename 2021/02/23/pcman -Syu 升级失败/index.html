pcman -Syu 升级失败

发表于2021-02-23|更新于2024-12-14
|浏览量:

msys2 failed retrieving file ‘mingw64.db
msys2 failed retrieving file ‘mingw32.db
无效或已损坏的数据库 (PGP 签名)

解决方法:
pacman -U http://repo.msys2.org/msys/x86_64/msys2-keyring-1~20210213-1-any.pkg.tar.zst
