hexo添加gitalk提示Network Error

发表于2025-01-12|更新于2025-01-12|blog
|浏览量:

问题：

Error: Network Error

解决办法：

路径：node_modules\hexo-theme-butterfly\layout\includes\third-party\comments

文件：gitalk.pug

需要使用科学手段，在gitalk.pug直接添加proxy: https://xxxxx/https://github.com/login/oauth/access_token

或者在_config.butterfly.yml添加：

option：

    proxy:

cf-worker

参考：
https://github.com/ymyuuu/Cloudflare-Workers-Proxy
